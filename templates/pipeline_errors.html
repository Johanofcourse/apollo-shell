What is this? Apollo Sentinel checks each utility's and NWS's live data feed on its own schedule (about every 15 minutes). Every so often a single check fails — a utility's server is briefly slow, a network request times out, and so on. Every one of those is caught automatically and recorded here, then the system simply tries again on its next scheduled check.

This list is not a live problem report. A handful of entries here, especially old ones or ones that only happened once, is normal and expected — it's the same kind of thing any website or app quietly recovers from. The main dashboard's pipeline-health strip is the place to check whether something is failing right now (it turns amber/red only when a source has failed repeatedly, recently). This page exists purely for transparency: a plain, complete record of what happened and when, in case anyone — including a client — ever wants to see it for themselves.

Each entry below is color-coded by how much attention it deserves: green means self-inflicted and harmless, amber means a normal, temporary hiccup on the data source's end, and red means it's worth a second look if it keeps recurring. The plain-language explanation is our best interpretation of the raw technical message underneath it - the raw message itself is always kept too, in case anyone wants to see exactly what the computer said.
